# Exercise 1: Live Coding - Filtering and Transforming with Lambdas

## Description
This exercise demonstrates how to use lambdas for filtering and transforming collections in Kotlin.

## Learning Objectives
- Use `filter()` to select elements based on a condition
- Use `map()` to transform elements
- Use `forEach()` to iterate and perform actions

## How to Run
```bash
cd Exercise-1-LiveCoding-Filtering-Transforming
kotlinc src/main/kotlin/Main.kt -include-runtime -d output.jar
java -jar output.jar
```

## Expected Output
The program filters numbers greater than 5, squares them, and prints the results.
