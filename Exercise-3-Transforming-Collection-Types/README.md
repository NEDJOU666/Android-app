# Exercise 3: Transforming Between Collection Types

## Description
This exercise demonstrates how to transform data between different collection types such as Lists and Maps in Kotlin.

## Learning Objectives
- Use `associateWith()` to create maps from lists
- Filter maps based on values
- Work with key-value pairs
- Group data using `groupBy()`

## Key Functions Used
- **associateWith()**: Creates a Map where keys are list elements and values are computed
- **filter()**: Filters map entries based on a condition
- **groupBy()**: Groups elements by a key function

## How to Run
```bash
cd Exercise-3-Transforming-Collection-Types
kotlinc src/main/kotlin/Main.kt -include-runtime -d output.jar
java -jar output.jar
```

## What It Does
Demonstrates various ways to transform lists into maps and perform operations on the resulting collections.
